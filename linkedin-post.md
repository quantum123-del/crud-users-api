# Post LinkedIn - Lancement du Projet API REST

---

## ✨ Version Française

🚀 **Nouveau projet : API REST de collecte de données utilisateurs !**

Je suis heureux de partager la réalisation de mon API REST développée avec Node.js et Express, connectée à Supabase pour la gestion de la base de données.

🎯 **Fonctionnalités implémentées :**
- ✅ Système complet de CRUD (Create, Read, Update, Delete)
- ✅ Authentification sécurisée avec Supabase Auth
- ✅ Protection des routes par token JWT
- ✅ Filtrage des données par âge
- ✅ Messages de confirmation pour chaque action
- ✅ Gestion des erreurs robuste

🛠️ **Stack technique :**
- Node.js + Express
- Supabase (Base de données + Auth)
- Architecture MVC (Controllers, Routers, Middlewares)

Ce projet m'a permis de mettre en pratique mes compétences en développement backend et d'approfondir mes connaissances sur les APIs REST et l'authentification.

🔜 Prochaine étape : Développement du frontend pour consommer cette API !
